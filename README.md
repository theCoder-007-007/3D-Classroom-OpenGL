# 3D Classroom Simulation using OpenGL

## Features
- First-person camera movement
- Mouse look
- Animated ceiling fan
- Lighting toggle
- Projector
- Classroom furniture
- Textured walls

## Controls
- W/A/S/D : Move
- Mouse : Look around
- L : Toggle lights
- Z/X : Camera roll
- ESC : Exit

## Technologies
- C++
- OpenGL
- GLUT
